# SpaceGame

This is a game I made using Python and turtle.

## How to run it ?

```
python3 game.py
```

## What is SpaceGame ?

SpaceGame is a game where you shoot aliens

you finish all the levels

and you kill the final boss

## Install package

On arch

```
sudo pacman -S tk
```

On Ubuntu

```
sudo apt-get install tk
```
